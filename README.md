Простой набор скриптов для переноса треков из liked между разными аккаунтами SoundCloud.

Доступ к API soundcloud не дает, поэтому берем его сами.

Краткий гайд:

1. Зайти на сайт soundcloud.
2. Открыть консоль разработчика и вставить туда содержимое [`soundcloud.js`](./soundcloud.js).
3. Посмотреть количество треков в плейлисте `liked`.
4. Вызвать `dumpLikes(likedCount)`.
5. Скопировать результат.
6. Выполнить шаги 1 и 2 с другого аккаунта.
7. Вызвать `loadLikes(copiedLikes)`.
